## 2026-02-14 - E2E Testing Research Complete

Completed research on E2E testing strategies for TUI applications (issue sidebar_tui-mgh). Cloned three key libraries to references/tui-testing/: (1) terminal-testlib (ratatui-testlib) - a comprehensive PTY-based test harness for ratatui apps, (2) expectrl - Rust implementation of Tcl's expect for automating interactive terminals, (3) rexpect - simpler expect-style API. **Recommendation: Use terminal-testlib as primary E2E testing tool.** It provides real PTY spawning via portable-pty (same lib we'll use for terminal emulation), keyboard event simulation (including Ctrl+Q), text assertions with wait_for_text(), and headless mode for CI. Created detailed summary at references/tui-testing/RESEARCH_SUMMARY.md with complete test patterns matching our specific requirements (layout verification, git status output, vi editing, backspace handling). One research issue remains: existing projects study (sidebar_tui-90c).
